## Access Tiers

The Public API uses `X-API-Key` authentication with three access levels:

| Tier | Key prefix | Permissions |
|------|-----------|-------------|
| **READ_ONLY** | `cxp_ro_...` | GET endpoints only |
| **STANDARD** | `cxp_std_...` | READ_ONLY + create/update operations |
| **FULL** | `cxp_full_...` | STANDARD + destructive and admin-grade operations |
