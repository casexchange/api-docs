---
title: "Current User"
description: "Retrieve the authenticated user profile"
---

<Endpoint method="GET" path="/api/v1/auth/me" />

## Response

```json
{
  "success": true,
  "data": {
    "id": "uuid",
    "email": "user@example.com",
    "firstName": "John",
    "lastName": "Doe",
    "role": "member",
    "firmId": "uuid",
    "firm": {
      "id": "uuid",
      "name": "Firm Name"
    },
    "isActive": true,
    "lastLogin": "2024-01-01T00:00:00Z",
    "createdAt": "2024-01-01T00:00:00Z"
  }
}
```

Use this endpoint immediately after login to confirm context, roles, and firm associations. It is also helpful for session dashboards or debugging token scopes.
