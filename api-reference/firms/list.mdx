<Endpoint method="GET" path="/api/v1/firms" />

Retrieve a paginated list of firms visible to your organization. Use this endpoint to discover firms for case referrals or to populate firm selection dropdowns.

## Authentication

<ParamField header="Authorization" type="string" required>
Bearer token obtained from login or OAuth flow. Format: `Bearer <token>`
</ParamField>

## Query Parameters

<ParamField query="page" type="integer" default="1">
Page number for pagination. Minimum: 1.
</ParamField>

<ParamField query="limit" type="integer" default="20">
Number of results per page. Range: 1-100. Default: 20.
</ParamField>

<ParamField query="search" type="string">
Search term to filter firms by name. Case-insensitive partial matching.
</ParamField>

<ParamField query="isInternal" type="boolean">
Filter by internal firm status. Restricted to System Admins. Default: `false`.
</ParamField>

<RequestExample>
```bash cURL
curl -X GET 'https://api.casexchange.com/api/v1/firms?page=1&limit=20&search=Johnson' \
  -H 'Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...'
```
</RequestExample>

<RequestExample>
```javascript Node.js
const params = new URLSearchParams({
  page: '1',
  limit: '20',
  search: 'Johnson'
});

const response = await fetch(`https://api.casexchange.com/api/v1/firms?${params}`, {
  headers: {
    'Authorization': `Bearer ${accessToken}`
  }
});

const data = await response.json();
console.log(`Found ${data.pagination.total} firms`);
```
</RequestExample>

## Additional Routes

- `GET /api/v1/firms/{id}` – Direct lookup by firm UUID.
- `GET /api/v1/firms/directory` – Curated list suitable for public browsing with advanced filters.

## Example: Find Firm for Referral

```javascript Node.js
async function findFirmForReferral(caseType, jurisdiction, accessToken) {
  const params = new URLSearchParams({
    page: '1',
    limit: '50'
  });

  const response = await fetch(`https://api.casexchange.com/api/v1/firms?${params}`, {
    headers: {
      'Authorization': `Bearer ${accessToken}`
    }
  });

  const { data, pagination } = await response.json();
  
  const matchingFirms = data.filter(firm => 
    firm.specialties.includes(caseType) &&
    firm.jurisdictions.includes(jurisdiction) &&
    firm.canSendReferrals
  );

  return matchingFirms;
}

const firms = await findFirmForReferral('personal_injury', 'CA', accessToken);
console.log(`Found ${firms.length} matching firms`);
firms.forEach(firm => {
  console.log(`${firm.name} - ${firm.contactEmail}`);
});
```

<Tip>
Use the `search` parameter to quickly find firms by name. Combine with pagination to browse large firm directories efficiently.
</Tip>
