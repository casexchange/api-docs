---
title: "List Firms"
description: "Retrieve all firms visible to your organization"
---

```http
GET /api/v1/firms
Authorization: Bearer <token>
```

## Query Parameters

- `page` – default 1
- `limit` – default 20, max 100
- `search` – firm name filter
- `isInternal` – boolean, restricted to System Admins

## Additional Routes

- `GET /api/v1/firms/{id}` – direct lookup by ID.
- `GET /api/v1/firms/directory` – curated list suitable for public browsing with advanced filters.
