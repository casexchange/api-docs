<Endpoint method="GET" path="/api/v1/firms/jurisdictions/list" />

Retrieve a simple list of supported jurisdiction codes. Use this endpoint for lightweight validation or simple dropdowns.

## Authentication

<ParamField header="Authorization" type="string" required>
Bearer token obtained from login or OAuth flow. Format: `Bearer <token>`
</ParamField>

<RequestExample>
```bash cURL
curl -X GET 'https://api.casexchange.com/api/v1/firms/jurisdictions/list' \
  -H 'Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...'
```
</RequestExample>

<RequestExample>
```javascript Node.js
const response = await fetch('https://api.casexchange.com/api/v1/firms/jurisdictions/list', {
  headers: {
    'Authorization': `Bearer ${accessToken}`
  }
});

const data = await response.json();
console.log('Supported jurisdictions:', data.data);
```
</RequestExample>

## Example: Validate Jurisdiction

```javascript Node.js
async function getJurisdictions(accessToken) {
  const response = await fetch('https://api.casexchange.com/api/v1/firms/jurisdictions/list', {
    headers: {
      'Authorization': `Bearer ${accessToken}`
    }
  });

  const { data } = await response.json();
  return new Set(data);
}

async function validateJurisdiction(jurisdiction, accessToken) {
  const jurisdictions = await getJurisdictions(accessToken);
  
  if (!jurisdictions.has(jurisdiction)) {
    throw new Error(`Invalid jurisdiction: ${jurisdiction}. Valid codes: ${Array.from(jurisdictions).join(', ')}`);
  }
  
  return true;
}

await validateJurisdiction('CA', accessToken);
console.log('Jurisdiction is valid');
```

<Tip>
Use the list endpoint (`/api/v1/firms/jurisdictions/list`) for lightweight validation and simple dropdowns. Use the detailed endpoint (`/api/v1/firms/jurisdictions`) to power UI copy or filters with friendly names.
</Tip>
