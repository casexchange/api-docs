<Endpoint method="GET" path="/api/v1/firms/case-types" />

Retrieve all available case types with their metadata. Use this endpoint to populate case type dropdowns or validate case type values before creating cases.

## Authentication

<ParamField header="Authorization" type="string" required>
Bearer token obtained from login or OAuth flow. Format: `Bearer <token>`
</ParamField>

<RequestExample>
```bash cURL
curl -X GET 'https://api.casexchange.com/api/v1/firms/case-types' \
  -H 'Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...'
```
</RequestExample>

<RequestExample>
```javascript Node.js
const response = await fetch('https://api.casexchange.com/api/v1/firms/case-types', {
  headers: {
    'Authorization': `Bearer ${accessToken}`
  }
});

const data = await response.json();
console.log('Available case types:', data.data);
```
</RequestExample>

## Response

<ResponseField name="success" type="boolean" required>
Indicates whether the request succeeded.
</ResponseField>

<ResponseField name="data" type="array" required>
Array of case type objects.

<Expandable title="Case type properties">
<ResponseField name="id" type="string" required>
Unique identifier for the case type. Format: UUID v4.
</ResponseField>

<ResponseField name="name" type="string" required>
Case type identifier. Use this value when creating cases.
</ResponseField>

<ResponseField name="description" type="string" required>
Human-readable description of the case type.
</ResponseField>
</Expandable>
</ResponseField>

## Example: Validate Case Type

```javascript Node.js
async function getCaseTypes(accessToken) {
  const response = await fetch('https://api.casexchange.com/api/v1/firms/case-types', {
    headers: {
      'Authorization': `Bearer ${accessToken}`
    }
  });

  const { data } = await response.json();
  return data.reduce((map, type) => {
    map[type.name] = type.description;
    return map;
  }, {});
}

async function validateCaseType(caseType, accessToken) {
  const caseTypes = await getCaseTypes(accessToken);
  
  if (!caseTypes[caseType]) {
    throw new Error(`Invalid case type: ${caseType}. Valid types: ${Object.keys(caseTypes).join(', ')}`);
  }
  
  return caseTypes[caseType];
}

const description = await validateCaseType('personal_injury', accessToken);
console.log(`Case type description: ${description}`);
```

<Tip>
Use the structured endpoint (`/api/v1/firms/case-types`) when you need descriptions or IDs for display purposes. Use the list endpoint (`/api/v1/firms/specialties/list`) for simple dropdowns that only need the case type names.
</Tip>
