## Users Endpoints

Base URL: `https://api.casexchange.com/api/public/v1`

---

### GET /users

Returns all users belonging to the API key's own firm. Supports pagination.

**Tier:** `read_only`

**Query parameters:**

| Parameter | Type    | Default | Description                  |
|-----------|---------|---------|------------------------------|
| `page`    | integer | `1`     | Page number (1-indexed).     |
| `limit`   | integer | `25`    | Results per page (max 100).  |

```bash
curl -X GET "https://api.casexchange.com/api/public/v1/users?page=1&limit=25" \
  -H "X-API-Key: cxp_ro_your_key_here"
```

### POST /users

Invites a new user to the caller's firm. An invitation email is sent to the provided address.

**Tier:** `full`

**Required fields:** `email`, `firstName`, `lastName`, `role`

**Valid roles:** `ADMIN`, `ACCOUNT_MANAGER`, `FIRM_ADMIN`, `MEMBER`, `VIEWER`

```bash
curl -X POST "https://api.casexchange.com/api/public/v1/users" \
  -H "X-API-Key: cxp_full_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "email": "priya.sharma@sterlinglaw.example.com",
    "firstName": "Priya",
    "lastName": "Sharma",
    "role": "MEMBER"
  }'
```

### PUT /users/{id}

Updates a user in the caller's firm. All body fields are optional -- include only the fields you want to change. Email must be unique across all users and is lowercased automatically.

**Tier:** `full`

**Optional fields:** `firstName`, `lastName`, `email`, `role`, `isActive`

```bash
curl -X PUT "https://api.casexchange.com/api/public/v1/users/f2a3b4c5-6789-0def-0123-456789abcdef" \
  -H "X-API-Key: cxp_full_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "firstName": "Dana",
    "lastName": "Kim-Torres",
    "role": "FIRM_ADMIN",
    "isActive": true
  }'
```

### DELETE /users/{id}

Deactivates a user in the caller's firm. Sets the user to inactive, clears security tokens, and revokes all active sessions. This action does not permanently delete the user record.

**Tier:** `full`

```bash
curl -X DELETE "https://api.casexchange.com/api/public/v1/users/a3b4c5d6-7890-1ef0-1234-567890abcdef" \
  -H "X-API-Key: cxp_full_your_key_here"
```

**Response `204`**

No content returned.
