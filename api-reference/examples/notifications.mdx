## Notifications Endpoints

Base URL: `https://api.casexchange.com/api/public/v1`

---

### GET /notifications/preferences

Returns the notification preferences for the caller's firm.

**Tier:** `read_only`

```bash
curl -X GET "https://api.casexchange.com/api/public/v1/notifications/preferences" \
  -H "X-API-Key: cxp_ro_your_key_here"
```

**Response `200`**

```json
{
  "data": {
    "emailOnNewReferral": true,
    "emailOnStatusChange": true,
    "emailOnDocumentUpload": false
  },
  "meta": {
    "requestId": "req_4e8f2a6b1c3d5079",
    "timestamp": "2026-03-31T16:00:00.000Z"
  }
}
```

---

### PUT /notifications/preferences

Updates the notification preferences for the caller's firm. All body fields are optional -- include only the fields you want to change.

**Tier:** `full`

```bash
curl -X PUT "https://api.casexchange.com/api/public/v1/notifications/preferences" \
  -H "X-API-Key: cxp_full_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "emailOnNewReferral": true,
    "emailOnStatusChange": true,
    "emailOnDocumentUpload": false
  }'
```

**Response `200`**

```json
{
  "data": {
    "emailOnNewReferral": true,
    "emailOnStatusChange": true,
    "emailOnDocumentUpload": false
  },
  "meta": {
    "requestId": "req_5f9a3b7c2d4e6180",
    "timestamp": "2026-03-31T16:00:05.000Z"
  }
}
```

### GET /notifications/emails

Returns all notification email addresses configured for the caller's firm.

**Tier:** `read_only`

```bash
curl -X GET "https://api.casexchange.com/api/public/v1/notifications/emails" \
  -H "X-API-Key: cxp_ro_your_key_here"
```

**Response `200`**

```json
{
  "data": [
    {
      "email": "intake@sterlinglaw.example.com"
    },
    {
      "email": "j.martinez@sterlinglaw.example.com"
    }
  ],
  "meta": {
    "requestId": "req_6a0b4c8d3e5f7291",
    "timestamp": "2026-03-31T16:00:10.000Z"
  }
}
```

---

### POST /notifications/emails

Adds an email address to the firm's notification list. The `email` field is required.

**Tier:** `full`

```bash
curl -X POST "https://api.casexchange.com/api/public/v1/notifications/emails" \
  -H "X-API-Key: cxp_full_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "email": "notifications@acmelaw.example.com"
  }'
```

**Response `201`**

```json
{
  "data": {
    "email": "notifications@acmelaw.example.com"
  },
  "meta": {
    "requestId": "req_7b1c5d9e4f6a8302",
    "timestamp": "2026-03-31T16:00:15.000Z"
  }
}
```
