Base URL: `https://api.casexchange.com/api/public/v1`

Returns all medical referrals linked to the caller's cases.

**Tier:** `read_only`

```bash
curl -X GET "https://api.casexchange.com/api/public/v1/medical-referrals" \
```

### POST /medical-referrals

Creates a new medical referral linked to a case. Required fields: `clientFirstName`, `clientLastName`.

**Tier:** `standard`

```bash
curl -X POST "https://api.casexchange.com/api/public/v1/medical-referrals" \
  -H "X-API-Key: cxp_std_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "baseCaseId": "a1b2c3d4-5678-9abc-def0-1234567890ab",
    "clientFirstName": "Jane",
    "clientLastName": "Doe",
    "clientEmail": "jane.doe@example.com",
    "clientPhone": "555-123-4567",
    "providerId": "c3d4e5f6-7890-1abc-def2-34567890abcd",
    "notes": "Medical referral for post-accident orthopedic evaluation",
    "dateOfInjury": "2026-01-15",
    "injuryDescription": "Knee injury from auto accident"
  }'
```

### PUT /medical-referrals/{id}

Updates an existing medical referral. All body fields are optional -- include only the fields you want to change.

**Tier:** `standard`

```bash
curl -X PUT "https://api.casexchange.com/api/public/v1/medical-referrals/a6b7c8d9-3456-7cde-f012-90abcdef0123" \
  -H "X-API-Key: cxp_std_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "clientPhone": "555-999-8888",
    "notes": "Updated evaluation notes -- patient also reports shoulder pain",
    "injuryDescription": "Knee and shoulder injury from auto accident"
  }'
```
